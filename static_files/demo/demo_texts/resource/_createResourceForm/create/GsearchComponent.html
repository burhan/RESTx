A new RESTful resource has been created

Here you see the confirmation that your resource was created successfully. You also receive the confirmation of the chosen resource name. This is important, since the actual name of the created resources is up to the server. The 'resource name' in the resource creation form is merely a suggestion.

Lastly, you get the URI of the RESTful resource you just created. That URI is all that a user of the new resource needs to know. Accessing it provides you with information about how to use the resource.
